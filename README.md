# 介绍
- earthui 是 earthsdk 所有功能的主要展示项目。
- earthsdk3-html 是 earthsdk 在html中使用示例。
- earthsdk3-vite 是 earthsdk 在使用vite搭建的项目中的使用示例。
- earthsdk3-wbpk 是 earthsdk 在使用常规vue搭建的项目中的使用示例。
- earthsdk3-openlayers 是earthsdk一个自定义引擎扩展和自定义对象扩展的示例，使用openlayers作为渲染引擎。

# 使用
 - 推荐使用 pnpm 和 vite 搭建的项目来集成 earthsdk ( 安装命令: npm install -g pnpm )
 - 任意目录运行 pnpm install 安装依赖
 - 对应目录下运行 pnpm dev/xxx
